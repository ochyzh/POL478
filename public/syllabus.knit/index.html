\bigskip \section*{Overview and Objectives} Real data, such as election results, location and type of terrorist attacks, or international trade flows, rarely come in nice, convenient to manage spreadsheets. Most data are not stored in spreadsheets at all. The task of collecting and cleaning the data is frequently upon the data analyst. Time-consuming and challenging as it is, this task need not be tedious and frustrating.

This course introduces students to the fundamental computational tools for cleaning, managing,n storing, analyzing, and presenting political data. The material will enable students to become proficient enough to actively implement the methods and tools in their own research.

\subsection*{Learning Outcomes} This course is designed as a series of weekly modules that build upon each other. Each module covers one or more state-of-the-art approaches to statistical analysis of network data. For each model covered, the objectives are that students will:

\begin{itemize} \item Learn how to deal with complex, messy, real data \item Use graphics to explore, understand, and present data \item Gain familiarity with basic data collection, storage and management \item Reshape data into the most convenient form for analysis or reporting \item Automate cleaning and analysis in R. \end{itemize}

\section*{Homework} Students will have a weekly homework (due before each class). For each homework, students will apply the methods covered that week to a dataset made available by the instructor. Each student will submit their homework as an R-script and a pdf file. Each homework will take 1–3 hours to complete.
